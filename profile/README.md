# SIAKAD MAS Aisyiyah Medan

Repositori ini berisi pengembangan Sistem Informasi Akademik berbasis web untuk MAS Aisyiyah Medan.

## Deskripsi Singkat

Sistem Informasi Akademik ini dikembangkan untuk membantu proses pengelolaan data akademik sekolah secara terintegrasi, meliputi pendaftaran murid baru, pengelolaan data murid, guru, kelas, jadwal pelajaran, absensi, nilai, pengumuman, serta profil sekolah.

## Repositori

### Backend API
Repository backend Laravel:

### Frontend Web
Repository frontend React + Vite:

## Fitur Utama

- Profil sekolah
- PPDB Online
- Login multi-role
- Dashboard administrator
- Dashboard kepala sekolah
- Dashboard guru
- Dashboard wali kelas
- Dashboard murid
- Pengelolaan data akademik
- Absensi murid dan guru
- Pengelolaan nilai
- Pengumuman sekolah
